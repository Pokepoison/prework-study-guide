HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hypertext Markup Language.
  • HTML is responsible for giving content structure.
  • Element refers to whole text from opening to closing tag.
  • Tag refers to only what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is responsible for the content's style
  • Inline CSS enables you to style an element dirctly in a HTML file, proving useful for quick and permanent, but less flexible changes.
  • Internal Style Sheet enables you to style an element directly in a HTML file, and apply style to multiple tagged elements at once.
  • External Style Sheet is a link to an outside CSS file where you can more elaborately edit the style.

Git

    \
  • Git is a version control system that enables a developer to keep track of different version of their code and also collaborate with other developers.
  • Version control is the practice of tracking changes in our code.
  • A repository is every project on GitHub. It is a folder for your project that contains all fo the file and their revision history.
  • A branch is a workspace where you can make changes that don't affect the live site.
  • Markdown is the way content filesa re formatted in Github. It is plain text that easily converts to HTML. (.md)
  • A pull request is asking for your changes on a branch to merge into another branch.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A : makes sure you're on the right feature branch.
  • git commit -m "commit message" : Commits changed file to repository.
  • git pull origin main : Make's sure current branch is up to date.
  • git push origin feature/file-name : Push up changes to repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A String is a variable that contains a sequence of text in quotation marks
  • A Boolean is a variable that has a True/False value.
  • An Array is a variable that contains multiple values in a single reference and requires quotation marks for each value listed.
  • An Array's values are all assigned a number, that starts at 0 and is called an Index.
  • A Number and a Boolean variable do not require quotation marks.
  • Control Flow is the order in which a computer executes a code in a file.
